# Vibe Coding → Agentic Engineering

**Skill level: Foundations**

## Definition
An AI-assisted development practice, term coined by Andrej Karpathy (Feb 2025), where a programmer describes intent in natural language and lets an LLM generate the code — the human's role shifts from writing syntax to guiding and testing. Karpathy's original framing was explicitly loose: paying little attention to the generated code itself.

## The shift toward agentic engineering
Simon Willison later coined "agentic engineering" (Feb 2026) to describe a more structured successor: building software with coding agents (Claude Code, Codex) that generate, execute, and iterate on code somewhat independently of turn-by-turn guidance.

## The risk
Unreviewed AI-generated code carries real architectural risk. Human code review and engineering judgment remain necessary regardless of how capable the model is.
